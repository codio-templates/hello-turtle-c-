## Modulo
**Modulo** is the mathematical operation that performs division but returns the remainder. The modulo operator is `%`.

![.guides/img/Modulo](.guides/img/Modulo.png)

```c++
int modulo = 5 % 2;
cout << modulo << endl;
```
