## Less Than

The `<` operator is used to check if one value is less than another value.

```c++
int a = 5;
int b = 7;
System.out.println(a < b);
```

## Less Than or Equal To

The `<=` operator is used to check if one value is less than or equal to another value.
```c++
int a = 5;
int b = 7;
System.out.println(a <= b;)
```
